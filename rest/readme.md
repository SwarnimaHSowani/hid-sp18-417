* download the eve git repo

* copy the settings..py and run.py to the eve folder
* execute 
      python run.py

      * http://127.0.0.1:5000/mysys/processor-name
      * http://127.0.0.1:5000/mysys/os-name
      * http://127.0.0.1:5000/mysys/sys-name
